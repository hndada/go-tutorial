# Go in an hour

**Go는 간결합니다. 한 시간이면 전체 문법의 2/3을 배울 수 있습니다.**
<p>&nbsp;</p>



파일 버튼을 누르면 파일 탐색기가 열립니다.
![파일 위치](https://i.imgur.com/jh0PLBr.png)

`replit`은 웹에서 처리되어 속도가 다소 느립니다. 원래 Go는 **빠릅니다**.
