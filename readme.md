# Go in an hour

**Go는 간결합니다. 한 시간이면 전체 문법의 2/3을 배울 수 있습니다.**



파일 버튼을 누르면 파일 탐색기가 열립니다.

`tutorial` 폴더에서 코드를 보며 순서대로 읽어보세요. 



각 코드 맨 아래에는 코드를 실행하기 위한 명령어가 적혀 있습니다. 

`Console` 창에 입력하면 실행됩니다.



replit은 웹을 통하여 처리되므로 실행 속도가 다소 느립니다. 원래 Go는 **빠릅니다**.
